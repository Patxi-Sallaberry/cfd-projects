## Projets

### 🛩️ [NACA 0012 — Simulation aérodynamique](./alpha_15deg)
**Statut : En cours**

Simulation CFD d'un profil NACA 0012 à différents angles d'attaque (α = 0°, 5°, 10°, 15°).
- Solveur : ANSYS Fluent 2026 R1
- Modèle de turbulence : k-omega SST
- Régime : stationnaire, incompressible
- Re ≈ 400 000 (V = 30 m/s, corde = 200mm)
- Objectif : polaire complète Cl/Cd = f(α)
